# Проект по БД: Получение данных о компаниях и вакансиях с сайта hh.ru и их загрузка в PostgreSQL

## Описание проекта
Проект включает в себя получение данных о компаниях и их вакансиях с сайта hh.ru, проектирование таблиц в БД PostgreSQL и загрузку полученных данных в созданные таблицы.
